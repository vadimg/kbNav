Terminology

  • labelize: put a shortcut label on a DOM element
  • actionize: associate a shortcut with an action

Quickstart

Download kbNav. You'll need jQuery 1.3.2 or higher. Put this in your <head> tag: 

<link rel="stylesheet" type="text/css" href="kbnav.css" />
<script type="text/javascript" src="jquery-1.7.2.js"></script>
<script type="text/javascript" src="kbnav.js"></script>
<!-- if you want to use kbNav on select boxes* -->
<script type="text/javascript" src="kbnav-select.js"></script>
*Note that you only need kbnav-select.js if you want to make your select boxes kbNav'd.

Buttons

To associate the shortcut 'a' with a button whose id is 'button' and put the appropriate label on it, put this code in your page: 

kbNav.register({
    a: { b: document.getElementById('button') }
});

Links

To associate the shortcut 's' with a link whose id is 'link' and put the appropriate label on it, put this code in your page: 

kbNav.register({
    s: { b: document.getElementById('link') }
});

Input Field

To associate the shortcut 'd' with a text box whose id is 'textbox' and put the appropriate label on a span with the id 'label', put this code in your page: 

kbNav.register({
    d: {
        l: document.getElementById('label'),
        a: document.getElementById('textbox')
    }
});
Label text:

The argument you pass to kbNav.register is a map of shortcuts -> actions/labels. 

shortcut: {
    l: (the DOM element to labelize),
    a: (the DOM element to actionize),
    b: (the DOM element to both labelize and actionize)
}

Look at the code for the widget examples for more examples.

More customization

If you want to add your own actionizers, take a look at kbnav-select.js. That file adds actionizers for select boxes.

If you want to use a custom labelizer, you can pass that in to the register function as the second argument, like this: 

kbNav.register({...}, {labelizer: function(shortcut, DOMelement){...}})
The labelizer takes two arguments, the shortcut (a string or number), and the target DOMelement.
